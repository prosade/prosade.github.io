搭建博客指北

三月 28, 2023

1. 准备前提工具：

  • Git
  • node.js

Git 安装步骤

  • 如果需要Git的更详细的步骤可移步至大佬mukes的详解。

  1. 点击Next进入Git的安装目录，选择Git的安装位置。

  2. 选择你所需要的附加功能，一般人按默认配置即可。

  3. 决定仓库的主干名字为“master”或者是“main”。

node.js安装步骤

  1. 安装node.js安装包

将Git和Github进行连接

  1. 在任意位置进行鼠标的右键点击Git Bash Here。

  2. 输入这三行代码设置用户名和邮箱

    其中第一二行分别是设置用户名和邮箱，而双引号中的分别是github的用户名与其绑定的邮箱。
    第三行为创建ssh密钥。输入创建ssh密钥后一直回车会在“C:\Users\用户名.ssh”下生成ssh密钥（ssh密钥为隐藏的项目，需要勾选“隐藏的项目”）。

  3. 用任意文档编辑程序打开“id_rsa.pub”并将其中的内容复制。

  4. 登录Github，点击右上角的用户头像，打开Settings

  5. 点击SSH and GPG keys

  6. 点击New SSH key，将复制的内容粘贴至Key文本框中。，Title的名字任意。最后点击Add SSH key。

    最后在Git Bush Here中输入ssh -T git@github.com验证连接。

安装hexo

  1. 新建一个文件夹用来存放blog文件，名字任意。在这个文件夹中右键，点击Git Bash Here。

  2. 输入以下代码完成安装Hexo。

    • npm install -g hexo-cli
    • npm init
    • npm install

  3. 执行以下代码验证是否完成安装，执行完下面两行代码后电脑回返回一个本地接口网址，用浏览器打开后即可看到本地部署的hexo。出现hexo页面时说明前面的步骤完成。

    1. hexo g //生成静态页面
    2. hexo s //启动预览

将本地Hexo部署到Github

  1. 安装hexo-deployer-git一键部署工具。
    输入代码npm install hexo-deployer-git --save完成安装。

  2. 修改配置

    1. 打开安装hexo时创建的存放Blog的文件夹，里面有配置文件_config.yml，用记事本之类的打开它，将配置文件拉到最下面，修改如下配置。
      • type: 后面写“git”
      • repository: 后面照着以下格式写：
        git@github.com:<git的用户名>/<git的用户名>.github.io.git
      • branch: 后写master。（大部分github现在都在用main，这和你之前创建github项目时选择的分支有关，查看你的github分支为master或者是main，这里就填写你的分支名）

  1. 在Git Bash Here中输入hexo d部署Hexo到Github上。

验证部署情况

在浏览器中输入网址[https://<github用户名>.github.io/]，之后就能看到你的博客了。

常用指令

  • hexo clean //清除缓存
  • hexo g //生成页面
  • hexo s //预览页面
  • hexo new “<文档名>” //新建文章
  • hexo d //上传博客
  • git clone <github的主题网址> themes/next //更换主题
